Sobre o projeto

Organização em Formato Mainframe

A Sala de Aula Virtual foi desenhada para estudantes da área de tecnologia que buscam foco e imersão. Ao invés de pastas de computador caóticas ou blocos de nota perdidos, o sistema estrutura seu aprendizado em uma hierarquia limpa: Cursos > Semestres > Disciplinas > Registros. Tudo é salvo instantaneamente no seu navegador usando LocalStorage e IndexedDB, garantindo total privacidade e velocidade.

Funcionalidades

O que o sistema oferece

Pomodoro Nativo

Gerencie ciclos de Foco e Descanso diretamente no cabeçalho. As cores mudam para alertar o modo atual (vermelho para foco, laranja para descanso) para máxima produtividade.

Armazenamento IndexedDB

O texto formatado, incluindo cores e destaques, é guardado usando a poderosa API do IndexedDB. Nenhuma informação é enviada a servidores, protegendo suas anotações.

Editor Estilo IDE

Um editor de anotações rico "WYSIWYG", incluindo suporte a formatações complexas, marca-textos personalizados, tarefas (checkboxes) e salvamento automático sem necessidade de clicar em botões.

Conhecimento Aplicado

Módulo de "Aplicação no Mundo Real" anexo a cada tópico. Força o aluno a documentar o problema real e a solução prática, retendo muito mais a matéria técnica de programação.

Identidade Visual Customizável

Acesse o menu do perfil para definir um Codinome de Operador, número de Matrícula Acadêmica e configurar o seu avatar com uma foto de preferência.

Navegação Dinâmica

Navegue entre cursos e semestres através de barras de seleção fixas no cabeçalho. Toda a UI se adapta na mesma tela sem recarregar a página, entregando a sensação real de um Single Page App.

Interface & Design System

Estética Cyber Hacker

O design evita a aparência de um aplicativo tradicional escolar. Com uma paleta rigorosamente estruturada em torno de pretos puros e vermelhos neons vibrantes, cantos quadrados para lembrar terminais UNIX antigos, e botões dinâmicos que geram "Glow" (brilho), o usuário se sente controlando um Command Center complexo.

Fluxos Lógicos

Arquitetura Operacional

Como os dados navegam dentro do sistema totalmente Client-Side:

Ação do Usuário Processamento Interno Onde é Guardado
Criar Nova Disciplina Adiciona um objeto na árvore JSON do Semestre atual e renderiza o Grid automaticamente. LocalStorage
Digitar Anotação no Editor Espera "1 segundo" (Debounce) após parar de digitar, criptografa e empacota o HTML rico. IndexedDB API
Clicar em "Pomodoro Play" Inicia o Timer, bloqueia cliques acidentais e acende a moldura vermelha do Header. Runtime (Variável na Memória)
Configurar Tempos do Pomodoro Atualiza as variáveis de sessão e recarrega os novos parâmetros para os próximos ciclos. LocalStorage (Perfil)
